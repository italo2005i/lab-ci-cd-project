# Lab CI/CD Project - Base Repository

Proyecto base para practicar CI/CD, liberación, mantenimiento correctivo y evolutivo.

## Estructura
- API simple en Node.js + Express
- Tests con Jest + Supertest
- GitHub Actions CI configurado
- Dockerfile para despliegues

## Cómo ejecutar localmente
1. Instalar dependencias:
   ```bash
   npm install
   ```
2. Ejecutar:
   ```bash
   npm start
   ```
3. Ejecutar tests:
   ```bash
   npm test
   ```
